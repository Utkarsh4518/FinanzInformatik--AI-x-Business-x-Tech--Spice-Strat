# BridgeFlow Plan

## One-Line Pitch

BridgeFlow turns messy project notes into structured execution plans, clearer communication, and fast handovers.

## Main Users

- Managers
- Business analysts
- Developers

## Main Problem Solved

Project information often starts incomplete, multilingual, and unstructured, which creates confusion between business and technical teams and slows delivery.

## MVP Features

- Manager project intake
- AI organizes messy input into tickets
- Kanban board as primary view
- Ticket table as secondary view
- Translation between business and technical language
- Progress summary for managers
- Handover generation when someone is unavailable
- Optional repo impact using local snapshot

## Out of Scope

- Auth
- Real Jira integration
- Live GitHub sync
- Vector database
- Custom model training
- Advanced scheduling engine

## Primary Demo Story

A manager pastes messy multilingual project notes about extending a loan calculator feature. BridgeFlow organizes the work into tickets, displays progress on a kanban board, explains technical implementation in business language, and generates a handover when one team member becomes unavailable.
